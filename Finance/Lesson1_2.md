# 72 法則
* 目的用於計算本金要過多少年才會翻倍
* 公式如下：
```
72 % R = T
```
1. 72 是一個神祕數字
2. R 是年利率，或是年投資報酬率
3. T 是本金翻倍的時間

* 如果 10 萬 放在銀行活存，年利率 1%，多久才會翻倍？
```
72 % 1 = 72
```
答案是 72 年

# 年利率
* 一年的存款利率，可從各家銀行告牌得知
    * 公式：複利息的本利和計算方程式是（本金 x （1+利率）時間）
    * 例如：
        * 存款 10,000 元
        * 年利率是 5%
        * 存放3年
        * 本利和是（10,000 x（1+0.05）3）=11,576.3元。


# 年投資報酬率
* 年投資報酬率是資金投入一年所得到的報酬率，可以分為兩類
1. 年平均報酬率
    * 算法是「總報酬率」% 「資金頭路的年數」

2. 年化報酬率，或稱為複利報酬率
    * 將每年獲利後的再投資利得也考慮進去
